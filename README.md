AWS Scalable Web Architecture (DevOps Project)

Overview
This project demonstrates a production-ready scalable web infrastructure using AWS services.

Architecture
Users → ALB → Target Group → Auto Scaling Group → EC2 → EBS

⚙️ Services Used
- EC2
- EBS
- AMI
- Launch Template
- Auto Scaling Group
- Application Load Balancer
- Target Groups

🔧 Features
- Automated server setup using User Data
- Dynamic scaling based on CPU utilization
- Load balancing across multiple instances
- Persistent storage using EBS
- Backup strategy using snapshots
